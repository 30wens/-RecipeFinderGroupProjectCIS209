# Recipe Finder Website

## Overview
The Recipe Finder Website helps users discover recipes based on the ingredients they already have. It displays results as interactive recipe cards that can be flipped to show ingredients, instructions, and nutritional details. Each card includes print and save options.

## Features
- Ingredient-based recipe search (validated using JavaScript)
- Interactive 3D recipe cards with flipping animations
- Contact/Feedback form validated with JavaScript
- Dynamic data handling and server communication using Node.js
- Responsive layout with header, footer, and navigation bar on every page

## Pages
- **Home Page:** Ingredient search bar, featured recipes  
- **Recipe Results Page:** Dynamic recipe cards, print/save buttons  
- **Contact/About Page:** Team info and contact form  

## Tools and Technologies
- HTML, CSS, JavaScript
- Node.js
- Draw.io (wireframes)
- GitHub for version control
